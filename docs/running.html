Pleroma

Getting Started - Installation

Run Pleroma on the first node:

pleroma

Run Pleroma on other nodes, specifying any accessible cluster IPs:

pleroma -f 192.168.178.1

Pleroma can also automatically handle local discovery:

pleroma --auto
